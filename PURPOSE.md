# Purpose

Errl Portal is a creative web portal application that combines interactive visual effects, studio tools, and component management into a unified interface.

## What It Is

A portal/launcher application that provides:
- Interactive portal interface with customizable visual effects
- Studio tools for visual editing and design
- Component catalog and management system
- Asset handling and management

## Architecture

Multi-app portal structure:
- Portal interface with navigation
- Multiple apps accessible from portal (Studio, Designer, etc.)
- Shared components and utilities
- Component catalog system

## Technology Stack

- React 19 - UI framework
- Vite - Build tool
- TypeScript - Type safety
- Monaco Editor - Code editing
- Fabric.js - Canvas manipulation
- Three.js / PixiJS - Graphics rendering
- GSAP - Animations
