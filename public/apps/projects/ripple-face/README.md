# Ripple Face

Errl face with a canvas ripple overlay. Click or drag to create expanding ripples.

## Files
- index.html, style.css, script.js
- assets/errl-face.svg

## How to run
- Direct file: open `index.html` in a browser (no server required)
- Via dev server: http://localhost:5173/apps/projects/ripple-face/

## Reduced motion
- Lower ripple growth and fade; ripples spawn on pointer down only.

## Browser support
- Modern evergreen browsers. Canvas 2D overlay only.

## Notes
- DPR-aware canvas sizing; resizes to container.
