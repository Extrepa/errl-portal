# Holographic Cursor Trail

Gradient dots follow the cursor and fade out using an opacity transition. No dependencies.

## Files
- index.html, style.css, script.js

## How to run
- Direct file: open `index.html` in a browser
- Via dev server: http://localhost:5173/apps/projects/holographic-cursor-trail/

## Reduced motion
- Lower density and faster fade; shorter lifetime.

## Browser support
- Modern evergreen browsers. No dependencies.
