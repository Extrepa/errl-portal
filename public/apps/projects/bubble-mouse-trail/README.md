# Bubble Mouse Trail

Lightweight DOM bubbles spawn along the cursor path and fade out with a keyframe animation.

## Files
- index.html, style.css, script.js

## How to run
- Direct file: open `index.html` in a browser
- Via dev server: http://localhost:5173/apps/projects/bubble-mouse-trail/

## Reduced motion
- Lower spawn rate and size; shorter lifetime.

## Browser support
- Modern evergreen browsers. No dependencies.
