Errl Hue Control System

Interactive demonstration of the hue rotation system for WebGL and CSS effects.

Basic Hue Controls

Adjust the hue, saturation, and intensity of effects:

Code:

ErrlHueController.setHue(180); // Blue shift
ErrlHueController.setSaturation(1.5); // More vibrant
ErrlHueController.setIntensity(0.7); // Subtle effect

Preset Themes

Quick presets for different color moods:

Code:

ErrlHueController.applyTheme('ocean');

Animation Controls

Animate hue rotation automatically:

Code:

ErrlHueController.startAnimation(1); // Speed 1
ErrlHueController.stopAnimation();

CSS Classes

Apply hue effects using CSS classes:

CSS Classes:

.hue-cycle-slow - 30s cycle
.hue-cycle-medium - 15s cycle
.hue-cycle-fast - 8s cycle
.hue-cycle-pulse - 4s pulse

Integration with Portal

In your main portal, the hue system automatically applies to:

  • WebGL Bubbles: Background floating bubbles with shader-based hue rotation
  • CSS Elements: Background layers, motes, HUD elements, navigation buttons
  • Overlay Effects: Shimmer and refraction overlays
  • Errl Character: Main character hue shifts (when not using WebGL)

Developer Panel: Press D in the main portal to access hue controls in the WebGL section.
