# Custom Orb Skins

Drop any PNG/JPG/WebP/SVG textures in this folder to make the HUD “Rotate Skins” button pick them up automatically. Vite bundles everything under `src/assets`, so you just need to add the files and rebuild/run `npm run dev`.

Accepted extensions: `.png`, `.jpg`, `.jpeg`, `.webp`, `.avif`, `.gif`, `.svg`.
